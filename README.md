Chesee

Профессиональный блокнот для заметок с графом связей.

Возможности
- Markdown редактор
- Интерактивный граф связей
- 5 тем оформления
- Ежедневные заметки
- GameDev Tracker

Установка
```bash
git clone https://github.com/NaDobrom/Chesee.git
cd Chesee
python -m venv .venv
.venv\Scripts\activate
pip install PyQt6
python main.py
